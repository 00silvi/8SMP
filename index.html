Materi Ajar

Konsep Dasar SPLDV

Pengertian SPLDV

Sistem Persamaan Linear Dua Variabel (SPLDV) adalah sistem yang terdiri dari dua persamaan linear dengan dua variabel yang sama.

Bentuk umum: a₁x + b₁y = c₁ dan a₂x + b₂y = c₂

Contoh SPLDV

1. 2x + 3y = 7 dan x - y = 1

2. 5x - 2y = 8 dan 3x + 4y = 10

Metode Penyelesaian

1. Metode Grafik

Menggambar grafik kedua persamaan dan mencari titik potongnya.

2. Metode Substitusi

Menggantikan salah satu variabel dengan ekspresi dari persamaan lain.

3. Metode Eliminasi

Menghilangkan salah satu variabel dengan operasi penjumlahan/pengurangan.

Contoh Penerapan

Masalah Sehari-hari

Contoh: Menentukan harga dua jenis barang jika diketahui kombinasi harga pembeliannya.

Langkah Penyelesaian

  1. Identifikasi variabel
  2. Buat model matematika
  3. Selesaikan SPLDV
  4. Interpretasikan solusi

Contoh Soal

Contoh 1:

Selesaikan sistem persamaan berikut:

$x + y = 10$
$x - y = 2$

Penyelesaian:

Kita dapat menggunakan metode eliminasi.

Tambahkan kedua persamaan:

$(x + y) + (x - y) = 10 + 2$

$2x = 12$

$x = 6$

Substitusikan nilai $x$ ke persamaan pertama:

$6 + y = 10$

$y = 4$

Solusi: $x = 6$, $y = 4$

Contoh 2:

Tentukan nilai $a$ dan $b$ yang memenuhi sistem persamaan:

$2a + 3b = 16$
$a - b = 2$

Penyelesaian:

Kita dapat menggunakan metode substitusi.

Dari persamaan kedua, kita dapatkan $a = b + 2$.

Substitusikan nilai $a$ ke persamaan pertama:

$2(b + 2) + 3b = 16$

$2b + 4 + 3b = 16$

$5b = 12$

$b = \frac{12}{5} = 2.4$

Substitusikan nilai $b$ kembali ke persamaan $a = b + 2$:

$a = 2.4 + 2 = 4.4$

Solusi: $a = 4.4$, $b = 2.4$

Contoh 3:

Harga 2 buah buku dan 3 buah pensil adalah Rp17.000. Jika harga sebuah buku adalah Rp500 lebih mahal dari harga sebuah pensil, berapakah harga sebuah buku dan harga sebuah pensil?

Penyelesaian:

Misal: harga buku = $b$, harga pensil = $p$

Model matematika:

$2b + 3p = 17000$

$b = p + 500$

Substitusikan persamaan kedua ke persamaan pertama:

$2(p + 500) + 3p = 17000$

$2p + 1000 + 3p = 17000$

$5p = 16000$

$p = 3200$

Substitusikan nilai $p$ kembali ke persamaan $b = p + 500$:

$b = 3200 + 500 = 3700$

Solusi: Harga buku = Rp3.700, Harga pensil = Rp3.200

Contoh 4: Metode Substitusi

Selesaikan sistem persamaan berikut menggunakan metode substitusi:

$y = 2x + 1$
$3x + y = 11$

Penyelesaian:

Substitusikan nilai $y$ dari persamaan pertama ke persamaan kedua:

$3x + (2x + 1) = 11$

$5x + 1 = 11$

$5x = 10$

$x = 2$

Substitusikan nilai $x$ kembali ke persamaan pertama:

$y = 2(2) + 1 = 4 + 1 = 5$

Solusi: $x = 2$, $y = 5$

Contoh 5: Metode Eliminasi

Selesaikan sistem persamaan berikut menggunakan metode eliminasi:

$4x - 2y = 12$
$x + 2y = 8$

Penyelesaian:

Tambahkan kedua persamaan untuk mengeliminasi $y$:

$(4x - 2y) + (x + 2y) = 12 + 8$

$5x = 20$

$x = 4$

Substitusikan nilai $x$ ke persamaan kedua:

$4 + 2y = 8$

$2y = 4$

$y = 2$

Solusi: $x = 4$, $y = 2$

Contoh 6:

Jumlah dua bilangan adalah 25, dan selisihnya adalah 7. Tentukan kedua bilangan tersebut.

Penyelesaian:

Misal: bilangan pertama = $p$, bilangan kedua = $q$

Model matematika:

$p + q = 25$

$p - q = 7$

Tambahkan kedua persamaan:

$(p + q) + (p - q) = 25 + 7$

$2p = 32$

$p = 16$

Substitusikan nilai $p$ ke persamaan pertama:

$16 + q = 25$

$q = 9$

Solusi: Bilangan pertama = 16, Bilangan kedua = 9

Contoh 7:

Sebuah persegi panjang memiliki keliling 48 cm. Jika panjangnya 6 cm lebih panjang dari lebarnya, tentukan panjang dan lebar persegi panjang tersebut.

Penyelesaian:

Misal: panjang = $l$, lebar = $w$

Model matematika:

$2l + 2w = 48$

$l = w + 6$

Substitusikan persamaan kedua ke persamaan pertama:

$2(w + 6) + 2w = 48$

$2w + 12 + 2w = 48$

$4w = 36$

$w = 9$

Substitusikan nilai $w$ kembali ke persamaan $l = w + 6$:

$l = 9 + 6 = 15$

Solusi: Panjang = 15 cm, Lebar = 9 cm

Contoh 8:

Selesaikan sistem persamaan berikut:

$0.5x + 0.2y = 2$
$x - y = 1$

Penyelesaian:

Kalikan persamaan pertama dengan 10 untuk menghilangkan desimal:

$5x + 2y = 20$

Dari persamaan kedua, $x = y + 1$. Substitusikan ke persamaan yang baru:

$5(y + 1) + 2y = 20$

$5y + 5 + 2y = 20$

$7y = 15$

$y = \frac{15}{7} \approx 2.14$

Substitusikan nilai $y$ kembali ke $x = y + 1$:

$x = \frac{15}{7} + 1 = \frac{22}{7} \approx 3.14$

Solusi: $x \approx 3.14$, $y \approx 2.14$

Contoh 9:

Di sebuah tempat parkir terdapat 15 kendaraan yang terdiri dari mobil dan motor. Jumlah roda seluruh kendaraan adalah 42. Berapa banyak mobil dan motor di tempat parkir tersebut?

Penyelesaian:

Misal: jumlah mobil = $m$, jumlah motor = $n$

Model matematika:

$m + n = 15$

$4m + 2n = 42$

Kalikan persamaan pertama dengan 2:

$2m + 2n = 30$

Kurangkan persamaan ini dari persamaan kedua:

$(4m + 2n) - (2m + 2n) = 42 - 30$

$2m = 12$

$m = 6$

Substitusikan nilai $m$ ke persamaan pertama:

$6 + n = 15$

$n = 9$

Solusi: Jumlah mobil = 6, Jumlah motor = 9

Contoh 10:

Selesaikan sistem persamaan berikut:

$\frac{1}{2}x + \frac{1}{3}y = 4$
$x - y = 2$

Penyelesaian:

Kalikan persamaan pertama dengan 6 untuk menghilangkan pecahan:

$3x + 2y = 24$

Dari persamaan kedua, $x = y + 2$. Substitusikan ke persamaan yang baru:

$3(y + 2) + 2y = 24$

$3y + 6 + 2y = 24$

$5y = 18$

$y = \frac{18}{5} = 3.6$

Substitusikan nilai $y$ kembali ke $x = y + 2$:

$x = 3.6 + 2 = 5.6$

Solusi: $x = 5.6$, $y = 3.6$

Latihan Soal

1. Selesaikan: x + y = 5 dan 2x - y = 4

Pembahasan:

Tambahkan kedua persamaan: 3x = 9 → x = 3

Substitusi x ke pers.1: 3 + y = 5 → y = 2

Solusi: x=3, y=2

2. Harga 3 buku dan 2 pensil Rp19.500. Harga 2 buku dan 3 pensil Rp16.000. Berapa harga 1 buku dan 1 pensil?

Pembahasan:

Misal: buku = x, pensil = y

3x + 2y = 19.500

2x + 3y = 16.000

Solusi: x=5.300, y=1.800

Harga 1 buku + 1 pensil = Rp7.100

Kuis Interaktif

1. Manakah yang merupakan SPLDV?

A. 2x + y = 5 dan x - y = 1
B. x² + y = 3 dan x - y = 2
C. 3x = 6 dan 2y = 4

Jawaban benar: A

SPLDV harus memiliki dua persamaan linear dengan dua variabel yang sama.

2. Penyelesaian dari x + y = 7 dan x - y = 3 adalah:

A. x=4, y=3
B. x=5, y=2
C. x=6, y=1

Jawaban benar: B

Tambahkan kedua persamaan: 2x = 10 → x=5

Substitusi: 5 + y = 7 → y=2
